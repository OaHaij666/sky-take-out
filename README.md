# 苍穹外卖 (Sky Take Out)

这是一个基于Java开发的后端练习项目，模拟外卖订餐系统的核心功能。

## 项目结构

项目采用分层架构设计，主要包含以下模块：

- **sky-common**：通用工具类和常量定义
- **sky-pojo**：数据模型层，包含实体类和数据传输对象
- **sky-server**：业务逻辑层和控制层，包含主要业务实现

## 技术栈

- 后端框架：Spring Boot
- 持久层框架：MyBatis
- 数据库：MySQL
- 其他：Spring Security、Redis等

## 功能模块

- 用户管理：用户注册、登录、信息管理
- 菜品管理：菜品分类、菜品信息维护
- 订单管理：下单、支付、订单状态跟踪
- 购物车管理：添加、删除、修改购物车商品
- 地址管理：用户收货地址管理

## 项目说明

本项目是一个后端练习项目，旨在帮助开发者熟悉Java Web开发、Spring Boot框架以及外卖系统的业务逻辑实现。项目提供了完整的后端API接口，可以与前端系统进行集成使用。

## 运行说明

1. 确保安装了JDK 1.8或以上版本
2. 确保安装了Maven
3. 配置数据库连接信息
4. 运行`mvn clean install`编译项目
5. 运行`mvn spring-boot:run`启动应用

## 注意事项

- 本项目仅用于学习和练习，请勿用于生产环境
- 项目中的部分功能可能需要额外配置，如支付接口、短信验证码等
